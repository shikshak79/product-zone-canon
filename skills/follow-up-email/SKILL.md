# Skill: Follow-Up Email

Write a follow-up email for a prospect using Product.Zone canon and user provided context.

## Inputs
- prospect record
- optional call transcript, notes, or prior email thread

## Canon context required
- canon/icp.md
- any canon items explicitly referenced in the input or needed to support the message

## Output
- A short personalized email
- 1 - 3 subject line options
- A brief noted `context_canon_used` listing canon items loaded
- Avoid making claims not supported by the canon or the prospect input
